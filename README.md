## 開き方

1. [https://nodejs.org/ja](https://nodejs.org/ja) を開く
2. 推奨版をクリックし、インストールする
3. 同意など色々クリックしてインストールする
4. VScodeでコードを開き、上部にあるターミナル>新しいターミナルをクリック
5. 下記コードを一行ずつ入力
6. 少し待つとlocalhost:3000などと出てくるのでGoogleChromeなどでURLにlocalhost:3000と入れる

```shell
npm i
npm run dev
```
